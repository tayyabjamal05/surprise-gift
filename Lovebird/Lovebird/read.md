# 🎁 Surprise Gift Store

Welcome to the **Surprise Gift Store** repository. This project is a premium web portal designed to showcase luxury gift boxes and bouquets with a cinematic feel.

### ✨ Features
- **Luxury UI:** Pearl white and gold aesthetic.
- **3D Interactive Experience:** Floating gift box animations.
- **Direct Order:** Integrated WhatsApp dispatch for quick purchases.
- **Fast Deployment:** Hosted on Firebase.

### 🛠️ Tech Stack
- HTML5 / CSS3 (Custom Animations)
- JavaScript (Vanilla)
- Firebase Hosting
